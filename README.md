# PowerBi_dashboard_projects

📊 Amazon Sales Analysis Dashboard (Power BI)

This project presents an interactive Amazon Sales Dashboard built using Power BI to analyze and visualize sales performance, customer behavior, and product trends.

🔍 Project Overview

The dashboard provides insights into:

Total sales, profit, and quantity sold
Category-wise and product-wise performance
Regional sales distribution
Monthly and yearly sales trends
Top-performing products and customers
